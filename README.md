## Dependencies ğŸ› 

### Frontend

1. **Yarn**:  
    ```bash
    yarn install
    ```
2. **React Router**:  
    ```bash
    yarn add react-router-dom
    ```
3. **React Testing Library**:  
    ```bash
    yarn add react-testing-library
    ```
4. **React-strap**:  
    ```bash
    yarn add react-strap
    ```
5. **Strap**:  
    ```bash
    yarn add strap
    ```
6. **Bootstrap**:  
    ```bash
    yarn add bootstrap
    ```
7. **Jest (Testing)**:  
    ```bash
    yarn add jest
    ```
8. **Animate.css**:  
    ```bash
    yarn add animate.css
    ```
